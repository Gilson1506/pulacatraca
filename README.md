# Pulacatraca

Sistema de gerenciamento de eventos e tickets.

## Funcionalidades

- Gerenciamento de eventos
- Sistema de tickets
- Check-in de participantes
- Integração com Supabase
- Interface moderna com React/TypeScript

## Tecnologias

- React + TypeScript
- Vite
- Supabase
- Tailwind CSS
